# 📚 Lista de Exercícios PHP

Projeto desenvolvido para praticar **conceitos básicos de PHP**, incluindo:

* Algoritmos sequenciais
* Estruturas condicionais (`if` e `switch`)
* Laços de repetição (`for`, `foreach`)
* Manipulação de formulários HTML
* Validação de dados

Todos os exercícios possuem **interface padronizada com CSS** e navegação através de um **menu principal**.

---

# 🚀 Tecnologias Utilizadas

* **PHP**
* **HTML5**
* **CSS3**
* **XAMPP / Apache** (ambiente de desenvolvimento)

# ▶️ Como Executar o Projeto

1. Instale o **XAMPP**
2. Coloque a pasta do projeto dentro de:

```
htdocs
```

Exemplo:

```
C:\xampp\htdocs\lista-php
```

3. Inicie **Apache** no XAMPP
4. Abra no navegador:

```
http://localhost/lista-php
```
